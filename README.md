# 📄 نظام التعرف الذكي على المستندات والأرشفة الآلية

**Smart Document Recognition & Automatic Archiving System**

## 🎯 نظرة عامة

تطبيق سطح مكتب يعمل **أوفلاين بالكامل** بمحرك ذكاء اصطناعي محلي لـ:

✅ استيراد وتحليل المستندات بجميع أنواعها  
✅ فهم محتوى المستندات باستخدام الذكاء الاصطناعي المحلي  
✅ استخراج أسماء ذكية بشرية للمستندات  
✅ تصنيف وأرشفة المستندات تلقائياً  
✅ نظام بحث ذكي وإدارة شاملة  

## 🔧 المتطلبات التقنية

- **Python**: 3.10 أو أحدث
- **نظام التشغيل**: Windows 10/11 (أساسي) + Linux + macOS
- **الذاكرة**: 8 GB حد أدنى (16 GB موصى)
- **مساحة القرص**: 10 GB (نماذج + تطبيق)
- **GPU**: غير مطلوب (يعمل على CPU)

## 📦 التثبيت

### 1. استنساخ المستودع
```bash
git clone https://github.com/Almoon866/Smart-archive.git
cd Smart-archive
```

### 2. إنشاء بيئة افتراضية
```bash
python -m venv venv

# على Windows
venv\Scripts\activate

# على Linux/macOS
source venv/bin/activate
```

### 3. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 4. تثبيت Ollama (للذكاء الاصطناعي المحلي)
```bash
# اذهب إلى https://ollama.ai/download
# وحمّل وثبّت Ollama

# ثم شغّل نموذج Qwen2
ollama pull qwen2:7b
```

### 5. إعداد متغيرات البيئة
```bash
cp .env.example .env
# عدّل .env إذا لزم الأمر
```

## 🚀 البدء السريع

```bash
python main.py
```

## 🎮 الميزات الرئيسية

### 1. استيراد ذكي
- سحب وإفلات للملفات
- اكتشاف نوع الملف الحقيقي
- التحقق من السلامة
- كشف التكرار

### 2. معالجة متقدمة
- استخراج النصوص من جميع الصيغ
- OCR عربي متقدم (PaddleOCR)
- معالجة الصور والمسح الضوئي

### 3. ذكاء اصطناعي محلي
- نموذج Qwen2 7B للعربية
- فهم السياق والمحتوى
- استخراج الأسماء والتواريخ
- التسمية الذكية البشرية

### 4. أرشفة ذكية
- تصنيف تلقائي
- هيكل أرشفة منظم
- حفظ البيانات الوصفية

### 5. بحث قوي
- بحث نصي كامل
- فلاتر متقدمة
- نتائج فورية

### 6. تقارير وإحصائيات
- رسوم بيانية
- إحصائيات شهرية
- تحليلات الاستخدام

## 📋 سير العمل

```
رفع الملف → التحقق من السلامة → اكتشاف النوع → استخراج المحتوى
    ↓
 هل يوجد نص؟ → OCR (إذا لزم) → تحليل النص
    ↓
إرسال لـ AI (Ollama) → استقبال التسمية الذكية
    ↓
عرض النتائج للمستخدم → موافقة/تعديل
    ↓
أرشفة وحفظ في قاعدة البيانات
    ↓
✅ تم بنجاح
```
